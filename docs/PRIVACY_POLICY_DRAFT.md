# Privacy Policy (Draft) — AzadiTunnel

**Last updated:** 2026-06-03

AzadiTunnel is an open-source VPN client. You import your own Psiphon JSON configuration; we do not ship private servers in the app.

## Data we store on your device

- Imported configuration (App Group shared container).
- Diagnostic event codes (no config body in logs).

## Data sent over the network

Determined entirely by your Psiphon configuration and the servers you connect to. Review Psiphon tunnel-core documentation for optional stats and server list fetch behavior.

## Data we do not collect centrally

AzadiTunnel has no proprietary backend in this repository.

## Contact

Provide a support email or issue tracker URL before App Store submission.
